## 简明vim

### 插入模式
- a -> 光标后插入
- i -> 光标前插入
- o -> 当前行后插入一行
- O -> 当前行前插入一行
- cw -> 替换当前光标到一个单词结尾的字符

### 简单的光标移动
- 0 -> 跳到行头
- ^ -> 跳到本行第一个不是blank的字符处
- $ -> 跳到行尾
- g_ -> 跳到本行最后一个不是blank的字符处
- /pattern -> 搜索pattern字符串

### 打开/退出/保存/改变文件
- :e <path/to/file> -> 打开文件
- :w -> 保存
- :x，ZZ和:wq -> 保存并退出
- :q! -> 退出不保存
- :qa! -> 强制退出所有的文件
- :bn，:bp -> 切换上一个或下一个文件
